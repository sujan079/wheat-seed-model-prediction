# wheat-seed-model-prediction

Project Description

A machine learning web application that predicts the type of wheat seed based on its physical characteristics using a trained classification model and a Streamlit interface.

How to Run the App

Clone the repository:

git clone https://github.com/sujan079/wheat-seed-model-prediction.git
cd weat-seed-model-prediction


Install the required dependencies:

pip install -r requirements.txt


Run the Streamlit application:

streamlit run app.py


Open the browser link shown in the terminal to use the app.
